# C Programming

- This course focuses on operating systems
  - primarily covering Linux
  - assumes that students have a reasonable understanding of the C programming language
- Concepts to Know
  - Structs, arrays, pointers, and reference types
  - File I/O
  - Use of command line parameters
  - Pass-by-reference and pass-by-value
  - Dynamic memory allocation using malloc()
  - Use of C libraries
- Other Helpful Skills Related to C
  - Debugging programs
  - Reading documentation
  - Iterative design
  - Good coding standards
