# String Functions

- [A Little C Primer/C String Function Library](https://en.wikibooks.org/wiki/A_Little_C_Primer/C_String_Function_Library)

```cpp
// Required declaration for string-function library
#include <string.h>
```

## Important String Functions

- _strlen()_
  - Get length of a string
  - not including the `NULL` character at the end
  - E.g.: `strlen('abs')`
- _strcpy()_
  - Copy one string to another
  - E.g.: `strcpy(s1, "abcdef")`
  - assumes that `s1` has enough space to store the final string
- _strcat()_
  - Link together (concatenate) two strings
  - E.g.: `strcat(s1, "ghijkl");`
- _strcmp()_
  - Compare two strings
  - E.g.: `strcmp(s1, "abcdefghijkl")`
  - returns a 0 for a successful comparison, and nonzero otherwise
- _strchr()_
  - Find finds the first occurrence of a character in a string
  - returns a pointer to the character if it finds it, and null if not
  - E.g.: `s1 = strchr(s1, 'g')`
- _strstr()_
  - Find string in string
- _strlwr()_
  - Convert string to lowercase
  - E.g.: `strlwr("ABC")`
- _strupr()_
  - Convert string to uppercase.
  - E.g.: `strupr("abc")`
