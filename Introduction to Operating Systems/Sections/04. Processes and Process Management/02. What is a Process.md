# What is a Process?

- The Operating System manages hardware on behalf of applications
- An application is just a program
  - stored on disk/flash/cloud/...
  - _static entity_
- Once an application is launched
  - loaded in memory
  - starts executing
  - becomes a process
  - _active entity_
  - same program launched more than once
    - multiple processes are created
    - executing the same program
    - having very different states
