# Managing Services with systemctl

- _systemd_ is used to manage services on the system
- _systemctl_ is the command line tool that manipulates and manages the services
- To find out what a service is doing
  - `systemctl status <service>`
- _Enable/Disable_ a service
  - If a service is enabled, it will automatically start on system boot
  - Will not start the service at present
  - `systemctl enable/disable <service>`
- _Start/Stop_ a service
  - `systemctl start/stop <service>`
