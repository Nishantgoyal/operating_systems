# Managing Services with SystemD

- Used by modern linux systems to manage their services
- The command line tool `systemctl` is used to
  - enable/disable a service
  - start/stop a service
  - check service status
  - etc...
- The service files for systemd can be scattered at multiple places
  - _/etc/systemd/system/_ folder
  - _/usr/lib/systemd/system_ folder
  - etc...
- Syntax: `systemctl <operation> <service>`
