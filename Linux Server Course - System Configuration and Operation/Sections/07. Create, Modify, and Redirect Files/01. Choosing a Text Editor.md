# Choosing a Text Editor

- It is an invaluable tool, since everything in Linux is text based
- Multiple options:
  - `nano`
    - easier to use
    - normal editing right away
    - commands mentioned below
    - to Exit - `^X` + some interaction
  - `vi`
    - steep learning curve
    - multiple modes
      - command mode
      - insert mode
      - visual mode

## VI

| Command      | Action              | Mode         |
| ------------ | ------------------- | ------------ |
| `:wq<enter>` | Save and Quit       | Command Mode |
| `:q!<enter>` | Quit without saving | Command Mode |
| `w<enter>`   | Save                | Command Mode |
| `i` or `a`   | Go to Insert Mode   | Command Mode |
| `<esc>`      | Go to Command Mode  | Insert Mode  |
| Typing Text  | Typing Text         | Insert Mode  |
