# STDIN, STDOUT and STDERR

- It is a way of getting information in and out
- Every application in Linux has three channels
  - _STDIN_
    - Redirect with `<` or `|`
    - `cat file.txt | grep item`
    - `grep item < file.txt`
  - _STDOUT_
    - Redirect with `>`
    - `ls > results.txt`
  - _STDERR_
    - Redirect with `2>`
    - For printing Errors
    - `ls ff 2>errors.txt`
