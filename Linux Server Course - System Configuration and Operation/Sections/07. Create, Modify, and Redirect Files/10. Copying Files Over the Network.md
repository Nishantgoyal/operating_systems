# Copying Files Over the Network

## `SCP`

```shell
# Copying file from remote to local
scp <user>@<host>:<port>:<path> <local_path>

# Copying file from local to remote
scp <local_path> <user>@<host>:<port>:<path>
```

## `rsync`

```shell
rsync -av <user>@<host>:<port>:<path> <local_path>
```

- `-a`: all files, recursive
- `-v`: verbose
