# Text Manipulation with command line tools

## `sort`

- sorts the output
- takes multiple options
  - ignore case: `-f`
  - numeric sort: `-g`
  - dictionary order: `-d`

```sh
sort file1.txt
ls | sort
```

## wordcount

- `wc`

```shell
wc file1.txt
#   4 4 27  file1.txt
```

- Output:
  - number of words
  - number of lines
  - number of characters
- supports multiple options to get required output:
  - character count: `-m`
  - line count: `-l`

## `cut`

- returns the character(s) at the specified positions in the file

```shell
cut -c 1 file1.txt
# c
# f
# t
# m
```

## `paste`

- joins two files

```shell
paste file1.txt file2.txt
# chicken lips
# fish    whiskers
# turtle  feathers
# nmonkey flippers
```
