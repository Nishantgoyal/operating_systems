# Editing Text with AWK & SED

- Two powerful text manipulation tools

## SED

- Stream Editor
- It allows you to edit files without interaction

```shell
cat file1.txt | sed s/monkey/dolphin/g
```

- It will
  - substitute
  - all occurances - cause of global flag`/g`
  - of word _monkey_
  - with the word _dolphin_

## AWK

- Data Extraction tool

```shell
awk '{ print $1 }' joined.txt
```

- it will print out the _first field_ of every line
