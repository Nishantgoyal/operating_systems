# Network Files - Ubuntu

- Ubuntu is based on debian
- The location of network configuration files has changed for the recent version of Ubuntu
- For Ubuntu16.04
  - configured via `/etc/network` directory
  - using the `/etc/network/interfaces` file
- For Ubuntu18.04 and above
  - configured via `/etc/netplan` directory
  - depending on the system, it will be having one of the config file
    - `01-network-manager.yaml`
    - `50-cloud-init.yaml` is it is cloud machine
  - To deploy change, run `sudo netplan apply`
