# Network Files - CentOS

- The location of all configuration files in CentOS is `/etc/sysconfig`
- For network specifically, the configuration files are in `/etc/sysconfig/network-scripts` folder
- There are many configurations file in the folder,
  - For each interface there is a configuration file:
    - _ifconfig-eth0_
    - _ifconfig-lo_
    - etc
  - also there are other configuration files
- Any changes to the network config files, require network restart
  `sudo systemctl network restart`
