# Finding Local Devices

- `dmesg`
  - It can be used to see how things are going in real-time
  - If a device is plugged in, _dmesg_ will log that and show the associated information
- `lsblk`
  - will show the block devices
- `lscpu`
  - provides all the information about the CPU in the system
- `lspci`
  - display all the PCI devices and controllers
- `lsdev`
  - it shows the memory map for all devices attached to the system
- `lsusb`
  - display all usb drives
- etc...
