# Managing RPM Packages

- RPM stands for _RedHat Package Manager_
- Packages can be managed by
  - _YUM_
    - de-facto standard
    - stands for _Yellowdog Updater Modified_
    - Handles dependencies
    - Updates repo information when installing/upgrading
  - _DNF_
    - newer
    - _DaNdiFied YUM_
    - In Fedora
    - will replace YUM
  - _RPM_
    - low level tool
    - actually handles packages
    - used by other package managers
