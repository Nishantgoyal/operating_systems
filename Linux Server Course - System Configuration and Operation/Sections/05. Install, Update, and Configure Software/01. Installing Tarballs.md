# Installing Tarballs

- Older method to installing softwares on linux
- Usually 3 steps:
  - extract
    `tar -zxvf <tarball>.tar.gz`
  - compile
    `config; make;`
    - resolve errors and dependencies
  - install
    `make install` or copy the binary to _/usr/local/bin_
- disadvantages
  - No update mechanism
