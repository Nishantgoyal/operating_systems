# Bootloaders - GRUB & GRUB2

- _Grand Unified Bootloader_
- GRUB2 is next iteration of GRUB
- It is just the way that
  - computers transitions
  - from BIOS of UEFI booting up conditions
  - into the Operating System
- It tells the computer
  - where are the partitions
  - etc

## GRUB

- Also called _GRUB legacy_
- Inside the boot grub folder (`/boot/grub`)
  - `menu.lst` and
  - `grub.conf` files are present
- It is difficult to modify
  - especially for the enduser
- displays the boot menu on loading

## GRUB2

- Inside the boot grub folder (`/boot/grub`)
  - `grub.cfg` is present
- Customizable via `/etc/default/grub` file
  - On any change, the following command needs to be executed
  - `sudo update-grub`
- More advanced then GRUB
  - It can boot from an ISO, USB disk
  - It can identify files based on their UUIDs, or device (like `/dev/sda`)
- hides the boot menu on loading
