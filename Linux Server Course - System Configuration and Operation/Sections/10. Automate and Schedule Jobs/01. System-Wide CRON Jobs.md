# System-Wide CRON Jobs

- Config Folder: _/etc/cron.d_
- Any file inside the directory is read by _CRON daemon_
- Format: `<CRON Fiels> <user> <application>`

## CRON Fields

- _minute_ _hour_ _day of month_ _month_ _day of week_
- e.g.,
  - `* * * * *`
    - Every minute
    - Every hour
    - Each day of month
    - Each month
    - Each day of week
  - `*/5 3 * * *`
    - Every _5th_ minute
    - At 3 AM
    - Each day of month
    - Each month
    - Each day of week
  - `2 4 13 7 2`
    - At _2nd_ minute
    - of _4th_ hour
    - On _13th_ of the month
    - in _July_
    - If it is a _Tuesday_
  - `0 6 * * 1-5`
    - At _0th_ minute
    - of 6th hour
    - Each day of month
    - Each month
    - _Monday_ to _Friday_

## CRON Folders

```shell
/etc/cron.daily
/etc/cron.hourly
/etc/cron.monthly
/etc/cron.weekly
```

- Any file inside the CRON folder is executed at every scheduled time
