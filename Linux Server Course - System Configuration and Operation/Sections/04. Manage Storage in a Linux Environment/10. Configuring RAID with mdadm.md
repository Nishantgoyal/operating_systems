# Configuring RAID with mdadm

- Creating a RAID device

```shell
# mdadm         -- tool to create RAID device
# /dev/md0      -- Name of the RAID device that will be created
# level         -- RAID level (0, 1, 5, 6, 01, 10, ..)
# raid-devices  -- No of disks to add to RAID
# /dev/sd*      -- list of disks to add to RAID
mdadm --create /dev/md0 --level=5 --raid-devices=4 /dev/sdb1 /dev/sdc1 /dev/sdd1 /dev/sde1
```

- To see details of RAID devices in the system
  `cat /proc/mdstat`
- To save the config of RAID devices, and create RAID device on boot
  `mdadm --detail --scan > /etc/mdadm/mdadm.conf`
- We can now create filesystem on the RAID device, and mount it
