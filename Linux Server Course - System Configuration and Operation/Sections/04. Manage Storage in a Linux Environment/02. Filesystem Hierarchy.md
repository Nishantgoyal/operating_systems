# Filesystem Hierarchy

- Everything in linux is on one giant file system, including
  - network mounts
  - different hard drives
  - USB drives
  - etc.
- The filesystem is like an interface
  - designed to interact with the kernel itself
- There is a root partition `(/)` in the linux filesystem
  - It is the base of the filesystem
- There are real filesystem
  - refers to the files and folders actually present on the local disk drive
- There are virtual filesystems
  - e.g., _/proc_, _/sys_
  - dynamically created
  - just a way to interact with the kernel
  - Do not contain actual files
- There can be remote systems
  - NFS drives, Samba, etc...
  - mounted on folders inside the root system
- Similarly there can be external disks attached
  - USB drives, Other Disks
  - mounted on folders inside the root system
