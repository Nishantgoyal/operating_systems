# LVM in a NutShell

- LVM stands for Logical Volume Manager
- Its like a software version of Storage Area Network
- It allows
  - join multiple physical disks into groups
  - use slices of storage from the group according to requirement
  - modifying the size of the storage on the fly
- It consists of following components
  - _physical volumes_
    - hard drives
    - RAID devices
    - partitions
    - `pvdisplay`
  - _volume groups_
    - combine physical volumes into volume groups
    - `vgdisplay`
  - _logical volumes_
    - divide the combines volume group into logical volumes
    - format with filesystem
    - mount on some folder path
    - `lvdisplay`
- long step process, but allows flexibility
