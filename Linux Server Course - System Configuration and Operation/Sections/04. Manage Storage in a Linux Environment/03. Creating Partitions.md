# Creating Partitions

- Organisational units on a hard drive
  - A disk is divided can be divided into partitions
- Each partition can be used for different things like
  - swap space
  - a particular folder on the system
  - etc
- _parted_
  - Partition Editor
- _gparted_
  - Graphical Partition Editor
- _fdisk_
- _lsblk_
  - list block devices
  - provides info about all the disks attached to the system
  - tree format output
- block device info is also available on `/proc/partitions` path
