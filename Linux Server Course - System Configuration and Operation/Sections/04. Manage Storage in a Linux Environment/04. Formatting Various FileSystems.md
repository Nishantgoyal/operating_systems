# Formatting Various FileSystems

- A filesystem in a disk is a way to manage data in a disk drive
- There are various types of filesystems
  - _ext_
    - Most Common
    - Mature
    - ext2/ext3/ext4
    - Later Versions Support Journal
  - xfs
    - Older
    - Used by CentOS
    - has its own set of tools
  - btrfs
    - Newer
    - Many features
  - dos
    - NTFS/FAT/VFAT
