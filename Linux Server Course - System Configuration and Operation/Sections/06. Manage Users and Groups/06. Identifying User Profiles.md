# Identifying User Profiles

- User Profiles are where initial settings are done
  - like aliases, path variables, etc
  - Can be done for each user as well as system wide
- Global Profiles
  - `/etc/environment` file
    - often sets up the path variable
  - `/etc/profile` file
    - run on a log-in shell
    - only executed one time on login
    - not executed on opening of each new shell
  - `/etc/profile.d` folder
    - contains shell scripts
    - all the shell scripts in included in the `/etc/profile` file
    - all the shell scripts are executed on login
  - `/etc/bashrc` or `/etc/bash.bashrc`
    - either one will be present depending on the OS
      - serve the same function
    - executed on each terminal session login
- User Profiles
  - `/home/user/.bashrc`
  - `/home/user/.profile` or `/home/user/.bash_profile`
