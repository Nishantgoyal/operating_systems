# Managing Local Users

## useradd

- Syntax: `useradd <options> username`
- `-d`: specify home directory
- `-s`: shell
- Issues:
  - Password need to be specified seperately
    - `passwd <username>`
  - specifying a home directory will not by default create it
    - need to specify `-w` to create the home directory
  - Need to remember a lot of flags

## adduser

- Syntax: `adduser username`
- interactive
- asks all the required inputs like password, shell, home directory, etc
- Set up the user more comprehensively

## userdel

- Syntax: `userdel <options> username`
- `-r`: removes the home directory and mail

## usermod

- Syntax: `usermod <options> username`
- `-s`: change shell
  - `usermod -s /bin/bash <user>`
