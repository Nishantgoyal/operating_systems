# Managing Group and Password Files

## Shadow System

- It consists of two files
  - _/etc/passwd_
    - consists of all users information
    - readable by all users
    - writable by root user
    - contains a placeholder _x_ in place of the password
  - _/etc/shadow_
    - contains the encrypted passwords
    - readable by user root and group shadow
    - writable by root user
- Similar system for groups:
  - _/etc/group_
  - _/etc/gshadow_

## Editing files

- To edit the _passwd_ file: `sudo vipw`
- To edit the _shadow_ file: `sudo vipw -s`
- To edit the _group_ file: `sudo vigr`
- To edit the _gshadow_ file: `sudo vigr -s`
